五一杭州旅游

发表于 2017-04-27 | 分类于 人生规划旅行

4月29日

  • 上午休息
  • 下午13:00出发
  • 第一站：杭州灵隐景区，（两地17公里，车程预计30分钟），13:30到达，主要景点：灵隐寺和飞来峰，游玩时间2小时，13:30-15:30
  • 第二站：梅家坞，（两地距离5.7公里，车程预计15分钟），农家气息，龙井茶诞生地，当地的农家菜，到处是茶叶，玩3-4小时，15:45-19:00
  • 晚上品杭州夜景
  • 回家

4月30日

  • 早起：7:00
  • 吃早餐出门：8:00
  • 出发去往西湖景区，行程大概50-60分钟，预计9:00到达
  • 西湖十景：
    苏堤春晓、曲苑风荷、平湖秋月、断桥残雪、柳浪闻莺、花港观鱼、雷峰夕照、双峰插云、南屏晚钟、三潭印月
  • 第一站：雷峰塔，玩1小时，9:00-10:00
  • 第二站：苏堤，（两地距离2.7公里，行程时间大概20分钟），玩1小时，10:20-11:30
  • 第三站：花港观鱼，（两地距离2.3公里，行程时间大概15分钟），玩一小时，吃中饭，此处吃饭的地方较多，11:45-13:00
  • 第四站：三潭印月，（两地距离2.8公里，行程时间大概15分钟），一元纸币的图案，可以上岛游览，玩2小时，13:00-15:00
  • 第五站：白堤+断桥残雪，（两地距离1.9公里，行程时间大概20分钟），这个地方非常浪漫，玩1小时，15:00-16:00
  • 第六站：南宋御街，（两地距离3.4公里，行程时间大概20分钟），古朴的商业街，类似南昌的珠宝街，玩1个半小时，16:20-18:00
  • 第七站：清河街坊，（两地距离500米，行程时间大概10分钟），商业步行街，各种小吃，玩1-2小时，晚饭，这里有张小泉剪刀、王星记扇子、酥油饼、定胜糕、龙须糖，18：10-20：00
  • 回家40分钟车程，预计到家21:00

4月31日

  • 早起：7:00
  • 吃早餐出门：8:00
  • 出发去往杭州客运中心站，行程大概50分钟，预计9点到达
  • 做杭州到乌镇的大巴，每15分钟一趟，最好提早买好票，可购买9:15或者9:25或者9:35的车
  • 大巴行程时间1.5个小时，预计到达时间11:00
  • 到了乌镇，一切自由发挥

  • 以下只做参考

  • 乌镇旅游景区分为东栅和西栅，其中以西栅最后著名。西栅由12座小岛组成，70多座小桥将这些小岛串连在一起，河流密度和石桥数量均为全国古镇之最。

  • 东栅景区：汇源当铺、访庐阁、皮影戏、翰林第、修真观、古戏台、茅盾故居、余榴梁钱币馆、木雕馆、蓝印花布染坊、公生糟坊、乌镇民俗风情馆、江南百床馆、传统作坊区、香山堂、拳船表演，逢源双桥（通济桥、仁济桥）等17处旅游景点。

  • 西栅景区：昭明书院、草木本色染坊、水阁和公埠石碑、水上戏台及评书场、 桥里桥、定升桥与定升糕、乌将军庙、月老庙、北湿地、京杭大运河、龙形田、元宝湖等所有景点。

  • 上午：东栅 → 进入景区 → 观看高杆表演 → 逢源双桥 → 财神湾 → 香山堂药店 → 江南百床馆 → 江南民俗馆 → 高公生酒坊 → 宏源泰染坊 → 走仁义桥 → 传统手工作坊区(糕点作坊、湖笔作坊、铜器作坊、布鞋作坊、刨烟作坊等)→ 江南木雕馆。中午： 在传统小吃一条街品尝风味小吃。

  • 下午：游览传统文化区及传统商铺区和水乡风景区 → 余榴梁钱币馆 → 茅盾故居、立志书院 → 古戏台 → 修真观 → 夏同善翰林第 → 皮影戏 → 传统商铺区 → 汇源当铺 →浙北分府(位于西栅景区边)

  • 景点总归纳：西栅、东栅、老邮局、宏源泰染坊、草木本色染坊、茅盾故居、江南百床馆、逢源双桥、公生槽坊、江南民俗馆、白莲塔寺、南栅、三寸金莲塔、天下第一锅、月老庙、昭明书院、乌镇桥里桥、西市河
